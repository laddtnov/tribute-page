## ✨ Features

* **Responsive Design:** Optimized for desktops, tablets, and mobile devices.
* **Typography:** Elegant font pairing using *Playfair Display* and *Lora*.
* **Visual Effects:** * 🏛️ Custom textured background for a "stone" feel.
    * 🖼️ Image hover effects (scale & grayscale transition).
    * ✒️ Stylized drop cap for a classic editorial look.
* **Semantic HTML:** Built with clean, accessible markup.

## 🛠️ Tech Stack

* **HTML5** - Structured with semantic tags.
* **CSS3** - Custom styling, Flexbox, and transitions.
* **Google Fonts** - Typography.
